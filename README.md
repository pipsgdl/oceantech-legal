## Por qué vive aquí

1. **Ocean Tech es el publisher** de las apps en App Store Connect (TeamID `8PB5VDRDUH`) y Google Play Console.
2. **Ocean Tech es el data controller** bajo LFPDPPP — opera la infraestructura (Supabase self-hosted), el Apple Developer Program y los servicios cloud.
3. **Los clientes (venues, comercios) son venue partners**, no responsables legales del tratamiento.
4. **Escalabilidad**: un mismo template se reutiliza para cada app nueva.
